Luas Segitiga

Rumus Luas Segitiga yaitu :
L = 1/2 x a x t

Di mana:

L = Luas
A = Panjang Alas
T = Tinggi

Keliling Segitiga

Rumus Keliling Segitiga yaitu :
K = S1 + S2 + S3

Di mana:

K = Keliling
S1 = Sisi A
S2 = Sisi B
S3 = Sisi C

Luas Jajar Genjang

Rumus jajar genjang yaitu :
L = a x t

Di mana:

L = LUAS
A = PANJANG ALAS
T = TINGGI

Keliling Jajar Genjang

Rumus Keliling Jaja Genjang yaitu :
K = 2 x (a + b)

Di mana:

K = Keliling
S1 = Sisi 1
S2 = Sisi 2
